# LaTeX OCR Desktop App

A simple desktop application for converting images of LaTeX equations to LaTeX code.

## Requirements

- Python 3.12+
- `pywebview`
- `pix2tex`
- `Pillow`
- `pyperclip`
- `torch`
- `torchvision`

## Installation

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
